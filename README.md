# Grape Soda Ranch Crafts Website Mock-up

Static website mock-up for **Grape Soda Ranch Crafts** with a warm, welcoming visual style and three pages:

- Home (`/index.html`)
- About Us (`/about.html`)
- Bio (`/bio.html`)

## Content sources

- Facebook profile URL: [https://www.facebook.com/becauzeof2](https://www.facebook.com/becauzeof2)
- Public profile metadata and public image were used directly from Facebook.

## Notes

- Facebook restricts anonymous scraping of full photo feeds. This mock-up includes the publicly accessible profile image and direct links to Facebook for full gallery browsing.
- Replace `assets/images/*` with additional approved client images to expand the gallery section.

## Run locally

Open `index.html` in a browser.

## Deploy

Push to GitHub and enable GitHub Pages (branch: `main`, folder: `/root`) to present to client.
